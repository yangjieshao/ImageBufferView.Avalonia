AGENTS.md
==========

# 角色规则
- 你是个资深c#程序员。
- 你必须提供高质量的代码建议，确保代码的可读性和可维护性。
- 必须添加 XML 中文注释
- 除代码外所有的交流/注释使用中文

# 代码规范
- 语言：C# 14，.NET 10
- 缩进：4 空格，禁用 Tab
- 在最新的c#语法中， 是可以用 [] 初始化数组的， 所以建议使用 new T[] 来初始化数组
- 尽可能使用 var 定义数据类型， 只有在无法从右侧推断出类型时才使用显式类型
- .axml 文件要求使用 XAML Styler 进行格式化
